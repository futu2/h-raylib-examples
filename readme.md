## Usage

```
git clone https://github.com/futu2/h-raylib-examples
cd h-raylib-examples
nix run
```
and choose example
```
Examples available:
1. basic window
2. input keys
3. input mouse
4. input mouseWheel
5. input gesture
6. camera 2d
7. camera 2d mouse zoom
8. camera 2d platformer
Input number to select example:
```
